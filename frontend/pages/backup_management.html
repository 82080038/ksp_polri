    üîÑ Setup Cron Job (Otomatis)

    Untuk backup otomatis setiap hari jam 2 pagi:

    1. Buka terminal dan jalankan: sudo crontab -e
    2. Tambahkan baris berikut:
    0 2 * * * cd /var/www/html/ksp_polri/backend && php scripts/backup_database.php >> logs/backup.log 2>&1

    Penjelasan: Backup akan berjalan setiap hari pukul 02:00. Log akan disimpan di logs/backup.log. Backup otomatis akan menyimpan file selama 30 hari.
